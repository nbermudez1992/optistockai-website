# OptiStockAI — "Cansado de Alegra?" Campaign
**Competitor Takeout Play | Created: 2026-03-19**

---

## 🎯 The Concept

Target businesses that are:
- Already using Alegra (or Siigo)
- Frustrated with limitations, lack of support, manual processes
- Not getting value from features they're paying for

**Positioning:** "You don't have to accept Alegra's limitations. We built OptiStockAI for businesses like yours that need MORE from their ERP — without switching."

## 🔍 Where to Find Frustrated Alegra Users

### Places to Monitor/Search:
1. **LinkedIn** — Search: "Alegra" + "frustrado" / "problemas" / "no funciona" / "soporte"
2. **Twitter/X** — Search: "AlegraColombia" complaints, "AlegraMexico"
3. **Facebook Groups** — "Empresas Colombian" / "PYMEs Mexico" / "Contadores Colombia"
4. **Google Reviews** — "Alegra" app reviews (play store, chrome store)
5. **Reddit** — r/colombia, r/mexico, r/emprendimiento
6. **Trustpilot** — "Alegra" reviews

### Outreach Trigger Strategy:
When you see someone publicly complaining about Alegra:
1. Like/engage with their post first (build rapport)
2. After 1-2 days, send a DM: "Vi tu post sobre Alegra... ¿Te ha pasado que [specific pain]?"
3. Offer OptiStockAI as the solution

### Google Alerts:
Set up alerts for:
- "Alegra problema"
- "Alegra no funciona"
- "alternativa a Alegra"
- "migrar de Alegra"
